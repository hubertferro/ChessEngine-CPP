# ChessEngine-CPP

A C++ chess engine project built to practice object-oriented programming, move validation, and game logic.

## Features

- Chess board representation
- Piece movement logic
- Move validation
- Turn-based gameplay
- Object-oriented design using C++ classes

## Technologies Used

- C++
- CLion
- Git
- GitHub

- Data structures
- Game logic
- Debugging
- Version control
